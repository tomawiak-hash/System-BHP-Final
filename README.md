# System-BHP-Final
Projekt pracy inżynierskiej: Inteligentny generator szkoleń BHP. Aplikacja w Streamlit wykorzystująca AI (Google Gemini) do automatyzacji tworzenia spersonalizowanych materiałów szkoleniowych oraz pełnej dokumentacji (.docx), takiej jak certyfikaty, dzienniki i wykazy. Personalizacja opiera się na analizie opisów zawodów.
